title: "Push"
order: 0
category: "spell"
description: "Push spell for SagaBorn D100"
---

*   Mana per Level: 1
*   Casting Time: 1 Standard Action
*   Range: 60’
*   Target: Single Target
*   Duration: Instant
*   Avoidance: Contested ACU vs target DEX or STR
*   Special: save is *difficult*, Critical: save is *very difficult*
*   Type: Utility

A magical force pushes the object away from the caster and causes bludgeoning damage. The target can be a person or an object.

Each additional mana can push a larger person or object (an additional 7 SIZ), push an additional 5’ backwards, and add 1d6 damage. If the target is a larger SIZ than the mana spent, it is not pushed, but it does take damage. Avoidance with a DEX or STR roll stops the creature from being pushed, and they only take 1/2 damage.

|  |  |  |  |  |  |  |  |
| :-: | :-: | :-: | :-: | :-: | :-: | :-: | :-: |
| **Mana** | **1** | **2** | **3** | **4** | **5** | **6** | **7** |
| **SIZ** | 7 | 14 | 21 | 28 | 35 | 42 | 49 |
| **Damage** | 1d6 | 2d6 | 3d6 | 4d6 | 5d6 | 6d6 | 7d6 |
| **Push Distance** | 5’ | 10’ | 15’ | 20’ | 25’ | 30’ | 35’ |

Spellcraft failure: The spell fizzles, and mana is lost.
