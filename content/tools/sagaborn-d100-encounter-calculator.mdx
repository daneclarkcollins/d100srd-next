function calculateCV() { const inputs = document.querySelectorAll("#tplInputs input"); const diff = parseFloat(document.getElementById("difficulty").value); let totalCV = 0; inputs.forEach(input => { const val = parseInt(input.value); if (!isNaN(val)) totalCV += val; }); const enemyCV = Math.round(totalCV \* diff \* 10) / 10; const results = document.getElementById("results"); results.innerHTML = \` <p><strong>Total Party CV:</strong> ${totalCV}</p> <p><strong>Recommended Enemy CV Budget:</strong> ${enemyCV}</p> <p>Build your encounter using monsters and tiers that add up to this value.</p> \`; }

### How to Use the SagaBorn Encounter Calculator

This tool helps StoryGuides quickly build balanced combat encounters using the SagaBorn BRP system. To use it:

1.  **Enter the [Talent Point Level](http://localhost:8200/glossary/talent-point-level/) (TPL)** for each hero in the party. Most starting characters begin with 3 TPL and gain 1 point per story arc.
2.  **Select your desired difficulty** from the dropdown. This will scale the encounter to be easy, balanced, or deadly based on the party’s combined TPL (used here as Combat Value, or CV).
3.  **Click "Calculate Encounter CV"** to see the total party CV and the recommended enemy CV budget.
4.  Use that number to choose a mix of monsters from the Creature Compendium. Combine Minions, Grunts, Champions, and Bosses until you match the budget.
    *   Remember: **Champions count as +0.5 CV**, and **Bosses as +1 CV**, due to their added defenses and health.
